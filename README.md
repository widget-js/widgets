## ▶️运行项目

1. 下载并运行桌面组件客户端
2. 克隆代码

```shell
git clone https://github.com/widget-js/widget.git
```

3. 到项目目录下载依赖

```shell
pnpm run install
```

4. 运行

```shell
pnpm run dev
```
